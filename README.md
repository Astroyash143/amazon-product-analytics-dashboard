# amazon-product-analytics-dashboard
Built an interactive Amazon Product Analytics Dashboard using Excel, Power BI, and DAX to analyze product pricing, discounts, ratings, and customer engagement.


## Overview
This project is an interactive Power BI dashboard built to analyze Amazon product data. The dashboard provides insights into product pricing, discounts, ratings, customer engagement, and category-wise performance.

---

## Tools & Technologies Used
- Microsoft Excel
- Power BI
- DAX (Data Analysis Expressions)

---

## Dataset Features
The dataset includes:
- Product Categories
- Actual Price
- Discounted Price
- Discount Percentage
- Product Ratings
- Rating Count
- Product IDs

---

## Key KPIs
- Total Products
- Average Rating
- Average Discount Percentage
- Total Reviews
- Average Actual Price
- Average Discounted Price
- Total Savings

---

## Dashboard Visuals
- KPI Cards
- Products by Category
- Average Rating by Category
- Price Comparison Charts
- Discount vs Rating Scatter Plot
- Rating Distribution
- Interactive Slicers & Filters

---

## Key Insights
- Some product categories offer high discounts while maintaining strong ratings.
- Higher customer engagement is observed in selected product categories.
- Discounted products often attract more reviews and customer attention.
- Pricing and discount strategies vary significantly across categories.

---

## Project Workflow
1. Data Cleaning using Excel
2. Data Transformation
3. DAX Measure Creation
4. Dashboard Design in Power BI
5. Business Insights Generation
